# Library_Management_System_Using_Java

This Java program manages library book details using a class and constructor. It stores information such as book name, author, type, price, and availability status. The program uses an array of objects to handle multiple books, methods to display book details, and the Scanner class to accept user input. Book details are entered in a comma-separated format, split using string operations, and displayed with a unique book code. It demonstrates object-oriented programming, arrays, loops, constructors, and basic input/output in Java.

This system is designed for real-time usage.

*School and College Libraries – To store and display book details and availability.
*Public Libraries – To manage book records and check whether a book is available or already issued.
*Digital Libraries – As a basic backend logic for managing book information.
*Bookstores – To maintain book inventory and pricing details.
